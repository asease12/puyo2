# ぷよぷよ風ゲーム v2.1 (6x12 / ぷよぷよ通ルール風)

GitHub に置いてそのままブラウザで動かせる、静的ファイル構成のぷよぷよ風ゲームです。

## 実装済み要件

- プロ風UI（スコア・連鎖表示、NEXT/NEXT2）
- 壁キック付き回転
- NEXT 2つ表示
- 4つ以上で消える連鎖
- 1マスずつ落ちる物理重力解決（連鎖中もステップ解決）
- `requestAnimationFrame` による 60FPS 前提ループ
- 連鎖時の簡易エフェクト
- 6列 x 12段 盤面
- ゲームオーバーライン相当の上段基準ライン表示

## 起動方法

静的ホスティングで動作します。

```bash
python3 -m http.server 8000
```

その後 `http://localhost:8000` を開いてください。

## 操作

- ← / → : 移動
- ↓ : ソフトドロップ
- Z / X (または ↑) : 回転
- Space : ハードドロップ
- P : ポーズ
- R : リスタート
